---
description: Run the mobile application using Expo Go
---

1. Ensure you have the **Expo Go** app installed on your physical device (Android or iOS).
2. Open a terminal and navigate to the mobile directory:
   ```bash
   cd mobile
   ```
3. Start the Expo development server:
   // turbo
   ```bash
   npx expo start
   ```
4. A QR code will appear in the terminal.
5. Scan the QR code:
   - **Android**: Use the scan tool inside the Expo Go app.
   - **iOS**: Use the system Camera app.
6. The application will bundle and launch on your device.
